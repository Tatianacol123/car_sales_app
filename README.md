# Proyecto Sprint 7 – Aplicación Streamlit al análisis de datos sobre Vehículos
Por medio de la presente aplicación web, podmeos visualizar datos de vehículos usados publicados en EE.UU, incluyendo gráficos interactivos que fueron diseñados con Streamlit y Plotly.

## Funcionalidad
- Visualizar un histograma del odómetro.
- Visualizar un histograma del año del vehículo.
- Analizar la relación entre el precio y el kilometraje mediante los gráficos de dispersión.
- Interactuar con botones y elementos de control (checkbox).

## Archivos del proyecto
vehicles_us.csv — Dataset del proyecto
notebooks/EDA.ipynb — Análisis exploratorio inicial
App.py — Código en la aplicación Streamlit
requirements.txt — Librerías necesarias


## Intrucción para la ejecución
1. Creación del entorno virtual
   python -m venv vehicles_env

2. Activación:
   Windows PowerShell:
   .\vehicles_env\Scripts\Activate.ps1

3. Instalación de dependencias:
   pip install -r requirements.txt

4. Ejecución:
   streamlit run app.py
